Projeto N-covid
###############

Esta página contém a documentação da plataforma N-covid. O propósito principal desta plataforma é fornecer um ambiente (site) que concentre recursos importantes para acompanhamento da evolução de pandemias virais, como visualização de dados da pandemia de diferentes naturezas e em diferentes níveis regionais, predição de tendencia das curvas e eventual sugestão de ações para reduzir os efeitos da pandemia. Em sua versão inicial, a platafoma é projetada para ser usada na pandemia da COVID-19.


Funcionalidades
---------------

- Agregação de dados
   - Update automático de dados das bases cadastradas;
   - Upload de dados relevantes
- Visualização de dados da pandemia
   - Curvas mais importantes para as regiões selecionadas
   - Vários níveis de granuralidade
- Treinar e Avaliar modelos de predição
   - várias estratégias de treinamento
   - métricas
   - métodos automáticos para avaliação
- Geração e visualização de predições
   - Modelos pré-treinados
   - métricas que indiquem a qualidade do modelo
- Processamento dos dados da pandemia
   - índices de correlação
   - limpeza dos dados
- Configuração dos parâmetros do modelo de predição
   - tipo, hyperparâmetros, entradas, dias de predição, etc..
- Simular efeitos que ações terão sobre os atributos


.. note::

   This project is under active development.
